# Azure Labs

Hands-on Azure labs completed while preparing for the Microsoft AZ-104: Azure Administrator certification.

## About

This repository documents my hands-on Azure learning through practical labs

## Technologies Used

- Microsoft Azure
- Azure Portal
- Azure Cloud Shell
- Azure CLI (Bash)
- Azure PowerShell
- Azure Virtual Networks (VNets)
- Resource Groups

---

## Lab Tasks Completed

- Retrieved an Azure Resource Group using Azure PowerShell.
- Created a Virtual Network using Azure CLI (Bash).
- Created a Virtual Network using Azure PowerShell.
- Verified deployed resources in the Azure Portal.
- Practiced navigating between Azure Portal, Bash, and PowerShell.

---

## Troubleshooting

During the Azure CLI deployment, I initially referenced multiple Resource Groups in a Bash variable, which caused the deployment to fail.

After identifying the issue, I corrected the variable to reference only the intended Resource Group and successfully deployed the Virtual Network.

Learning to troubleshoot deployment issues is an important part of Azure administration.
